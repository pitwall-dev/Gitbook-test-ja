# 👓 インスタントルックアップ

## Overview

ブラウザ拡張機能は、サイトやツールをオンデマンドで検索するための強力なアドオンです。

## 機能

ブラウザ拡張機能には以下の機能が備わっています。

1. ルックアップ
   * ブラウザ内に表示されるテキスト、たとえばIPアドレスにもとづいて検索。
2. グループ検索
   * キーを使って複数のサイトを検索。
3. スクリーンショット
   * エビデンスを残すために、スクリーンショットを取得。
